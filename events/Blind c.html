On 18 th & 19 th March'13

" Blind C "

C programming in blank screen

It is a Blind Programming event in which participants will write down a simple C program while monitor is switch off.


General Rules:


  1. Maximun number of participant equal to 1 per team. i.e. individual participant.
  2. Monitor will be turned off.
  3. Code should be logically comparable with given problem statement.
  4. Result will be based time, least number of error and logic.
  5. A participant will qualify for next round if his/her program execute properly for each test case. And is logically correct.
  6. If no one qualifies then result will be based on rule 4.

Entry fee : Rs. 150 per candidate.
